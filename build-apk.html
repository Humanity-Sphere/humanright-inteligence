Human Rights Intelligence Mobile App

APK-Erstellung für Android

Diese Anleitung führt Sie durch den Prozess der Erstellung einer installierbaren APK-Datei für Ihre Android-Geräte.

Hinweis: Sie benötigen ein kostenloses Expo-Konto und die EAS-CLI, um diesen Prozess abzuschließen.

Voraussetzungen

  • Ein Expo-Konto (kostenlos)
  • Node.js und npm auf Ihrem Computer
  • EAS-CLI: npm install -g eas-cli

Schritt-für-Schritt-Anleitung

1. Bei Expo anmelden

eas login

Geben Sie Ihre Expo-Anmeldedaten ein, wenn Sie dazu aufgefordert werden.

2. Projekt konfigurieren (nur bei erstmaliger Ausführung)

cd mobile-app
eas build:configure

3. APK erstellen

cd mobile-app
eas build -p android --profile preview

Dieser Prozess kann 10-15 Minuten dauern, da die APK in der Expo-Cloud erstellt wird.

4. APK herunterladen

Nach Abschluss des Build-Prozesses erhalten Sie einen Link zum Herunterladen der APK-Datei. Sie können die APK auch in Ihrem Expo-Dashboard finden.

5. Auf Android-Gerät installieren

Übertragen Sie die APK-Datei auf Ihr Android-Gerät und installieren Sie sie.

  • Möglicherweise müssen Sie die Installation aus unbekannten Quellen in den Einstellungen aktivieren
  • Navigieren Sie zu Einstellungen > Sicherheit > Unbekannte Quellen

Alternative: Mit Expo Go testen

cd mobile-app
npx expo publish

Nach der Veröffentlichung können Sie die App in der Expo Go App auf Ihrem Mobilgerät unter "Projects" finden.

Zu Expo gehen Ausführliche Anweisungen

QR-Code für Expo Go (lokale Entwicklung)

Wenn Sie sich im selben Netzwerk befinden, können Sie auch den QR-Code verwenden, um die App in Expo Go zu öffnen:

cd mobile-app && npx expo start

Der obige Befehl startet einen lokalen Entwicklungsserver und generiert einen QR-Code zum Scannen.

Über das Human Rights Intelligence Projekt

Die Human Rights Intelligence App unterstützt Menschenrechtsverteidiger mit fortschrittlichen KI-gestützten Werkzeugen für Dokumentation, Analyse und Berichterstattung.

Funktionen umfassen:

  • Dokumentenanalyse mit KI-Unterstützung
  • Intelligente Inhaltserstellung
  • Mustererkennungssystem für Menschenrechtsverletzungen
  • Sicherheits- und Zusammenarbeitstools
  • Rechtliche Unterstützung und Bildungsmaterialien
  • Integration mit Smart Glasses
