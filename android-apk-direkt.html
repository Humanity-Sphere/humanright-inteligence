APK Download

Sie können die Human Rights Intelligence App direkt als APK-Datei herunterladen und auf Ihrem Android-Gerät installieren.

APK HERUNTERLADEN
Hinweis: Die APK-Datei ist etwa 50MB groß. Sie benötigen eine stabile Internetverbindung für den Download.

Installationsanleitung

  1. Laden Sie die APK-Datei herunter, indem Sie auf den blauen Button oben klicken
  2. Öffnen Sie die heruntergeladene Datei auf Ihrem Android-Gerät
  3. Wenn Sie gefragt werden, erlauben Sie die Installation aus "unbekannten Quellen"
    • Dies finden Sie unter Einstellungen > Sicherheit > Unbekannte Quellen
    • Auf neueren Android-Versionen werden Sie direkt gefragt
  4. Folgen Sie den Anweisungen auf dem Bildschirm, um die Installation abzuschließen
  5. Nach der Installation können Sie die App direkt von Ihrem Startbildschirm aus öffnen

Funktionen der App

  • Dokumentenanalyse: Scannen und analysieren Sie Dokumente mit KI-Unterstützung
  • Rechtliche Unterstützung: Erhalten Sie rechtliche Hinweise zu Menschenrechtsfällen
  • Kampagnenplanung: Erstellen und verwalten Sie Kampagnen
  • Offline-Modus: Arbeiten Sie auch ohne Internetverbindung
  • Smart Glass Integration: Verbinden Sie sich mit unterstützten Smart Glasses
  • Sprachsteuerung: Bedienen Sie die App mit Sprachbefehlen
