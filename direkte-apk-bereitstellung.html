Direkte Erstellung einer APK-Datei

Für die Erstellung einer funktionsfähigen APK gibt es mehrere Möglichkeiten:

Option 1: Vorkompilierte APK

Wir haben eine fertige APK-Datei für Sie bereitgestellt. Sie können diese direkt herunterladen und auf Ihrem Android-Gerät installieren.

APK direkt herunterladen
Hinweis: Sie müssen in den Android-Einstellungen "Installation aus unbekannten Quellen" aktivieren, um die APK installieren zu können.

Option 2: Mit Expo EAS Build

Wenn Sie die App selbst bauen möchten, können Sie den Expo EAS Build-Dienst verwenden. Hier ist eine Schritt-für-Schritt-Anleitung:

  1. Erstellen Sie ein kostenloses Konto bei Expo
  2. Installieren Sie die Expo CLI: npm install -g eas-cli
  3. Melden Sie sich an: eas login
  4. Navigieren Sie zum Projektverzeichnis und führen Sie aus: eas build -p android --profile preview
  5. Warten Sie, bis der Build abgeschlossen ist (ca. 10-15 Minuten)
  6. Laden Sie die fertige APK-Datei von der angegebenen URL herunter

Alternative: APK über Github herunterladen

Eine weitere einfache Option ist, die APK direkt über GitHub zu beziehen:

  1. Besuchen Sie das GitHub-Repository der App
  2. Laden Sie die neueste Version der APK-Datei herunter
  3. Übertragen Sie diese auf Ihr Android-Gerät
  4. Öffnen Sie die Datei auf Ihrem Gerät und folgen Sie den Installationsanweisungen

Installation auf dem Android-Gerät

So installieren Sie die heruntergeladene APK-Datei:

  1. Öffnen Sie die Einstellungen Ihres Android-Geräts
  2. Gehen Sie zu "Sicherheit" oder "Biometrie und Sicherheit"
  3. Aktivieren Sie die Option "Installation aus unbekannten Quellen" oder "Unbekannte Apps installieren"
  4. Öffnen Sie die heruntergeladene APK-Datei in Ihrem Dateimanager
  5. Folgen Sie den Anweisungen auf dem Bildschirm, um die Installation abzuschließen
Wichtig: Nach der Installation können Sie die Berechtigungen für "Installation aus unbekannten Quellen" wieder deaktivieren, um die Sicherheit Ihres Geräts zu gewährleisten.
